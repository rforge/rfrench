Translation of R in French:
---------------------------
Since R version 2.1.0, the program is internationalized
(see http://www.r-project.org). A French translation for R is available
since version 2.2.1.

The French translation team is composed by (in alphabetic order):

Traduction:
Philippe Grosjean <phgrosjean@sciviews.org>
Frédéric Lehobey <Frederic@Lehobey.net>
Emmanuel Paradis <paradis@isem.univ-montp2.fr>
Jean Thioulouse <jthioulouse@biomserv.univ-lyon1.fr>

Documentation:
Yves Brostaux <brostaux.y@fsagx.ac.be>
Emmanuel Paradis <paradis@isem.univ-montp2.fr>

Proof-reading:
Santos Carvajal-Gonzalez <Santos.Carvajal-Gonzalez@pfizer.com>
Vincent Goulet <vincent.goulet@act.ulaval.ca>
Ted Harding <ted.harding@nessie.mcc.ac.uk>

Please, send any comment or suggestion about the French
translation of R directly to them. Many thanks.

Note also that an English-French dictionnary is available from this site
in the file RfrenchDictionnary.txt. R-specific terms are listed there with
their French translation as it is used here. We advise to use the same
translations all over any French document related to R. If you wish to
discuss one of these translations, please, contact the persons listed
above by email.
